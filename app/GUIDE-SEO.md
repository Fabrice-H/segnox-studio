## 🚀 ÉTAPES POUR ÊTRE PREMIER SUR GOOGLE

### PHASE 1 : Configuration immédiate (Aujourd'hui)

#### Étape 1 : Créer Google Analytics 4

1. Va sur https://analytics.google.com
2. Clique **"Commencer la mesure"**
3. Nom du compte : `SEGNOX Studio`
4. Nom de la propriété : `segnox-studio.com`
5. Fuseau : `Afrique/Abidjan`
6. Devise : `XOF`
7. Copie l'**ID de mesure** → `G-XXXXXXXXXX`

#### Étape 2 : Créer Google Tag Manager

1. Va sur https://tagmanager.google.com
2. Crée un compte `SEGNOX Studio`
3. Crée un conteneur `segnox-studio.com` (Web)
4. Copie l'**ID** → `GTM-XXXXXXX`

#### Étape 3 : Ajouter les variables d'environnement

Crée `.env.local` dans le dossier `app/` :

```env
NEXT_PUBLIC_GTM_ID=GTM-XXXXXXX
NEXT_PUBLIC_GA_ID=G-XXXXXXXXXX
```

#### Étape 4 : Google Search Console

1. Va sur https://search.google.com/search-console
2. Ajoute la propriété `https://segnox-studio.com`
3. Méthode de vérification : **Balise HTML** ou **DNS**
4. Une fois vérifié, copie le code de vérification
5. Ajoute dans `.env.local` :

```env
NEXT_PUBLIC_GOOGLE_SITE_VERIFICATION=ton_code
```

#### Étape 5 : Soumettre le Sitemap

1. Dans Search Console, va dans **Sitemaps**
2. Ajoute : `https://segnox-studio.com/sitemap.xml`
3. Clique **Envoyer**
